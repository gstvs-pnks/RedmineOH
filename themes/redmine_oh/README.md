# RedmineOH

Status: early development.

Design direction: an original macOS-influenced productivity interface.

Apple assets and proprietary icons are not included.

Theme work must not modify Redmine core.

Existing themes are not a source to copy.

Current stylesheet entry point: `themes/redmine_oh/stylesheets/application.css`

Manual activation: Administration -> Settings -> Display -> Theme -> RedmineOH

Runtime verification is performed manually by the project owner.
